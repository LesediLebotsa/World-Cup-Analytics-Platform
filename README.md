# World Cup Analytics & Prediction Platform

A software engineering project that combines football analytics, machine learning, data engineering and backend development.

The platform uses historical international football data and World Cup data to generate analytics, compare teams, and predict tournament outcomes.

## Planned Features

* Historical football analytics
* Team comparison dashboard
* REST API
* Rule-based prediction engine
* Machine learning prediction engine
* User authentication
* Password reset functionality
* Rate limiting
* Automated ETL pipelines
* Docker deployment
* CI/CD pipeline
* PostgreSQL database

## Technology Stack

* Python
* Flask
* PostgreSQL
* SQLAlchemy
* Pandas
* Streamlit
* Plotly
* Scikit-Learn
* Docker
* Pytest
